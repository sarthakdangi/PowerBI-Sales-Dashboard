# PowerBI-Sales-Dashboard
Power BI sales Analysis Dashboard Project . 
## Project Overview
The Sales Analysis Dashboard is an interactive Power BI solution designed to provide comprehensive insights into sales performance across multiple business dimensions including Period, Segment, Country, Product, Units Sold, Net Sales, Profit, and Profit Margin.
The dashboard enables business stakeholders, sales managers, and analysts to track KPIs, identify trends, evaluate profitability, and make data-driven decisions through dynamic visualizations and drill-down capabilities.

## Business Objective
The primary objective of this dashboard is to:

Monitor overall sales performance.
Track profitability across countries and products.
Analyze sales trends over time.
Identify top-performing segments and products.
Evaluate discount impact on revenue.
Enable stakeholders to make informed business decisions.

## Dashboard Features:

KPI Cards with trend indicators
Interactive filters
Discount Band Slicer
Data Table Summary
Navigation Menu
Export Functionality
